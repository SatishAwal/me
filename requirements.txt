# React Portfolio Project - Requirements and Setup Guide
# ===================================================

# Node.js and npm Requirements
# ----------------------------
# Node.js version: 16.0.0 or higher (recommended: 18.x or 20.x)
# npm version: 8.0.0 or higher
# 
# To check your versions:
# node --version
# npm --version

# Core Dependencies
# -----------------
react@^18.3.1                    # React library for building user interfaces
react-dom@^18.3.1               # React DOM rendering for web browsers
react-scripts@5.0.1             # Create React App scripts and configuration

# Development Dependencies
# -----------------------
@testing-library/jest-dom@^5.17.0      # Custom Jest matchers for DOM testing
@testing-library/react@^13.4.0         # React testing utilities
@testing-library/user-event@^13.5.0    # User event simulation for testing

# Additional Libraries
# --------------------
emailjs-com@^3.2.0              # Email service integration for contact form
gh-pages@^6.2.0                 # GitHub Pages deployment
web-vitals@^2.1.4               # Web performance metrics

# Browser Support
# ---------------
# Production browsers:
# - >0.2% (browsers with >0.2% market share)
# - not dead (not officially dead browsers)
# - not op_mini all (not Opera Mini)

# Development browsers:
# - last 1 chrome version
# - last 1 firefox version
# - last 1 safari version

# Installation Instructions
# ========================

# 1. Install Node.js and npm (if not already installed)
#    Download from: https://nodejs.org/

# 2. Clone the repository
#    git clone <repository-url>
#    cd portfolios

# 3. Install dependencies
#    npm install

# 4. Start development server
#    npm start

# 5. Build for production
#    npm run build

# 6. Deploy to GitHub Pages
#    npm run deploy

# Available Scripts
# ================
# npm start          - Start development server (http://localhost:3000)
# npm run build      - Build app for production
# npm test           - Run test suite
# npm run eject      - Eject from Create React App (irreversible)
# npm run predeploy  - Build app before deployment
# npm run deploy     - Deploy to GitHub Pages

# Project Structure
# ================
# src/
# ├── components/    - React components
# ├── styles/        - CSS files
# ├── assets/        - Images and other assets
# ├── App.js         - Main App component
# ├── index.js       - Entry point
# └── index.css      - Global styles

# Environment Setup
# ================

# Deployment
# ==========
# GitHub Pages: npm run deploy
# Netlify: Connect repository and build command: npm run build
# Vercel: Connect repository and auto-deploy
# AWS S3: Upload build folder to S3 bucket

# Troubleshooting
# ===============
# If npm install fails:
# - Clear npm cache: npm cache clean --force
# - Delete node_modules and package-lock.json
# - Run npm install again

# If build fails:
# - Check for syntax errors in components
# - Verify all imports are correct
# - Check browser console for errors

# If deployment fails:
# - Ensure repository is public (for GitHub Pages)
# - Check build output for errors
# - Verify homepage URL in package.json

# Development Tips
# ===============
# - Use React Developer Tools browser extension
# - Enable ESLint for code quality
# - Use Prettier for code formatting
# - Test on multiple browsers
